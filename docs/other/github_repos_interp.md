- **[activation_oracles](https://github.com/adamkarvonen/activation_oracles)** — Extracting secrets/goals/emotions from LLM hidden states via linear probes
- **[model-organisms-for-EM](https://github.com/clarifying-EM/model-organisms-for-EM)** — LoRA analysis, steering, activation-based interpretability (Qwen2.5-14B, Llama-3.1-8B)
- **[rl-rewardhacking](https://github.com/ariahw/rl-rewardhacking)** — RL training interventions using probes as reward-hacking monitors (LeetCode tasks)
